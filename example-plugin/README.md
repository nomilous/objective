# Example plugin.

```
# run in one console

../bin/objective

# it loads the objective from objective.js

```

```
# run in another console

../bin/objective --attach

# it attaches to the objective exposing a repl

# can view the running process variables, eg objective.*

# enter 'grabPrompt' to grab the objective prompt

# enter 'grab' to grab the objective console and debug output

# enter 'free' to release the console and debug output

```

```
# start objective with debugger on full blast

DEBUG=* ../bin/objective

```
